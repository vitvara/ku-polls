## Installation
Create virtual enviroment on this github directory<br>
`python -m venv venv`

Install package:<br>
`pip install -r requirement.txt`

Django migrations<br>

`python manage.py makemigrations`<br>
`python manage.py migrate`

Runserver<br>
`python manage.py runserver`
